🧪 StayOS Rooms Page - Button Test

📋 Testing Checklist

  • Open: http://localhost:3000/rooms?hotel=star7
  • Select Check-in date (must be today or future)
  • Select Check-out date (must be after check-in)
  • Select number of guests from dropdown
  • Select room type from dropdown
  • Click "Check Availability" button

🔍 Expected Behavior

When you click "Check Availability":

  1. If dates NOT selected: Shows warning "⚠️ Please select both check-in and check-out dates"
  2. If check-in is in past: Shows warning "⚠️ Check-in date cannot be in the past"
  3. If check-out <= check-in: Shows warning "⚠️ Check-out date must be after check-in date"
  4. If all valid:
    • Shows success message "✅ Showing available rooms for your dates!"
    • Smooth scrolls to rooms section
    • Displays active filters summary bar

Room Images:

Each room should display a different, beautiful hotel room image from Unsplash

Look for the photo count badge (📷 X photos) on rooms with multiple images

Filter Buttons:

Click on room type buttons (All Rooms, Standard Room, Deluxe Room, etc.)

The active filter should be highlighted in blue

Active filters summary should appear showing your selections
